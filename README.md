# Project Name: Server Backend

This is the backend server for my Bagrut engineering project. It is built using **Node.js** and **Express**, following a clean layered architecture.

## 🛠️ Project Structure
* **`server.js`** - The main entry point that configures and starts the server.
* **`Router/`** - Handles HTTP requests and maps them to the appropriate endpoints.
* **`Logic/`** - Contains the core business logic and data processing rules.
* **`DAL/` (Data Access Layer)** - Direct communication with the database.

## 🚀 How to Run Locally
1. Clone this repository to your computer.
2. Open your terminal in this folder and install dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   node server.js
   ```
